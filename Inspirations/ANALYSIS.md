## Part 2: Cross-Site Pattern Synthesis

### Color Strategy Patterns

#### Dark vs. Light Split
| Scheme | Count | Sites |
|--------|-------|-------|
| **Dark** | 8 (53%) | Linear, Neon, Resend, Framer, Raycast, Supabase, Inngest + parts of Mercury |
| **Light** | 7 (47%) | Stripe, Vercel, Cal.com, PostHog, Clerk, Dub, Anthropic, Mercury |

**Insight:** The split is nearly even. Developer tools lean dark. Products targeting broader business audiences lean light.

#### Background Color Categories
| Category | Colors | Sites |
|----------|--------|-------|
| **Pure Black** (`#000`) | `#000000` | Neon, Resend, Framer |
| **Near-Black** | `#07-#12xxxx` | Linear (`#08090A`), Raycast (`#07080A`), Supabase (`#121212`), Inngest (`#0C0A09`) |
| **Pure White** | `#FFFFFF` | Stripe, Dub |
| **Cool Off-White** | `#F4-#FA` | Vercel (`#FAFAFA`), Cal (`#F4F4F4`), Clerk (`#F7F7F8`) |
| **Warm Off-White** | cream/parchment | Anthropic (`#F0EEE6`), PostHog (`#EEEFE9`) |

**Key Takeaway:** Avoid medium grays as backgrounds. Go decisively dark or decisively light. Warm off-whites create a distinctive, premium feel that differentiates from the tech default.

| **Cool Palette** | Blues, purples, greens | Stripe, Vercel, Neon, Resend, Clerk, Supabase |

## Part 3: Actionable Design Recommendations

### For Dark Theme Landing Pages
```
Background: #000000 to #121212 (pure black to dark gray)
Text: #FAFAFA to #FFFFFF
Primary accent: ONE vibrant color (green, blue, or purple work best)
Button primary: Brand color bg + black or white text
Button secondary: Dark gray (#242424-#28282C) bg + light text + subtle border
Border radius: Pill (9999px) or medium (8-10px)
Font: Inter for body, custom/premium for headings
```

### For Light Theme Landing Pages
```
Background: #FFFFFF or #F7F7F8 (cool) or #F0EEE6 (warm)
Text: #0A0A0A to #171717
Primary accent: Purple, blue, or black buttons
Button primary: Brand color or black bg + white text
Button secondary: White bg + brand color text + subtle border
Border radius: Match primary button style
Font: Inter or Geist for body, optional custom heading
```

### Universal Typography Rules
```
H1: 48-72px (safe range), up to 110px for dramatic effect
H2: 32-56px
Body: 14-18px (16px is safest default)
Line height: 1.5-1.7 for body text
Font weight: 600-700 for headings, 400 for body
```

### Button Sizing Reference
```
Primary: padding 12-16px vertical, 24-32px horizontal
Font size: 14-16px
Min height: 40-48px
```

### Color Palette Construction
1. **Choose a base:** Pure black, near-black, pure white, or warm off-white
2. **Pick ONE brand color:** A saturated hue that contrasts with the base
3. **Derive grays:** Pull from the same temperature (cool grays for cool palettes, warm grays for warm)
4. **Text hierarchy:** Use opacity or slightly desaturated versions of your text color for secondary text
5. **Borders:** 1-2 stops lighter than background for dark themes, 1-2 stops darker for light themes

### Differentiating Strategies
To stand out from the Inter-on-dark-background default:
- **Use warm tones** (Anthropic, PostHog) — terracotta, cream, gold
- **Use serif fonts** (Anthropic) — scholarly, trustworthy
- **Use a custom font** (Stripe, Mercury, Framer) — branded, premium
- **Use sharp corners** (Inngest) — bold, angular, distinctive
- **Use unusually large or small type** (Framer at 110px, PostHog at 30px)
- **Use asymmetric border-radius** (Anthropic) — subtle but memorable
